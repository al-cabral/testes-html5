Canvas - Exemplo 5: apagando área retangular.

Conteúdo alternativo para navegadores que não suportam canvas.

JavaScript

<script>
window.onload = function() {
var canvas = document.getElementById('quadro');
var ctx = canvas.getContext('2d'); 
	ctx.strokeStyle = '#000';
	ctx.lineWidth = 2;
	ctx.fillStyle = '#060';
	ctx.fillRect(20,30, 300,150);
	ctx.strokeRect(18,28, 304,154);
ctx.clearRect(50,60,180,100);
}
</script>

CSS

	canvas {border:1px solid #ccc;}

HTML

	<canvas id="quadro" width="400" height="200">
		<p>Conteúdo alternativo para navegadores que não suportam canvas.</p>
	</canvas>
