Capítulo 5

Atributo dir

Demonstração

Essa página integra o conjunto de páginas exemplo do livro HTML5 do Maujor.

Foi criada para demonstar o atributo global dir

Clique os botões a seguir para definir os atributos ltr, rtl e auto para o elemento body dessa página

JavaScript

<script>
window.onload = function() {
var botao = document.getElementsByTagName('button')
var pg = document.getElementsByTagName('body')

	botao[0].onclick = function(){
		pg[0].setAttribute('dir', 'ltr');
	}
	botao[1].onclick = function(){
		pg[0].setAttribute('dir', 'rtl');
	}
	botao[2].onclick = function(){
		pg[0].setAttribute('dir', 'auto');
	}
}
</script>
